# Python-Copilot-course

A console-based Rock Paper Scissors Lizard Spock game in Python.

## Game Rules


- Scissors decapitate lizard
- Scissors cuts paper
- Paper covers rock
- Rock crushes lizard
- Lizard poisons Spock
- Spock smashes scissors
- Lizard eats paper
- Paper disproves Spock
- Spock vaporizes rock
- Rock crushes scissors

## Run the game

If the file is executable:

```bash
./rpsls
```

Or with Python:

```bash
python3 rpsls
```

Enter one of: `rock`, `paper`, `scissors`, `lizard`, `spock`.
Type `q`, `quit`, or `exit` to stop playing.
